Cryptonize

A platform to encrypt and decrypt your files.

Steps to Encrypt your File

To encrypt a file, enter a password and drop the file to be encrypted into the dropzone below.
The file will then be encrypted using the password and then you can save the encrypted file to your system.

ENCRYPTION

Enter your Password or Key

(minumum length eight characters, make sure it is strong!)


Enter your Password again to confirm


Drag and drop the file to be encrypted into this dropzone, or click here to select file.

Steps to Decrypt your File

Decrypt your file using the password that was previously used to encrypt the file.
After the file is decrypted, you can save the decrypted file to your system.

DECRYPTION

Enter your Password or Key


Drag and drop file to be decrypted into this dropzone, or click here to select file.
